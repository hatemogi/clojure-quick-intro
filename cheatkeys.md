# ProtoREPL 치트시트
^, B - 블럭 실행
^, D - 문서 표시
^, s - 선택영역 실행
^, K - 레플 지우기
ESC - 현재 인라인 결과 지우기
⎇ ⌂ C - 화면에 표시된 결과값 지우기
^, P - Pretty Print
^, N - list ns vars
^, ⌂ B - 탑 블럭 실행
^, O  - 코드 추적

# Atom
cmd \ 트리뷰 여닫기
cmd-shift + 폰트 키우기
cmd-shift - 폰트 줄이기
cmd L - 라인 선택
cmk K, cmd 방향 - 페인이동
^x o - 다른페인
^ cmd Z - zen toggle
^ cmd F - full screen
